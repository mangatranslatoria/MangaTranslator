User Guide

A practical guide for installing Manga Translator Studio, running your first translation and understanding the most important settings.

1. Install the app

Download the Windows installer, run it, then open Manga Translator Studio from the Start menu or desktop shortcut.

2. Start your first translation

  • Choose the manga image folder.
  • Choose the output folder.
  • Select the target language.
  • Pick a manga profile or use Generic Manga.
  • Click Start Translation.

3. LM Studio setup

Start LM Studio, load your preferred local model and enable the local server at http://127.0.0.1:1234/v1. Then select the same model in the app settings.

4. OCR engines

Use EasyOCR for most translated scans and Latin text. Use MangaOCR for Japanese manga when available. If OCR is wrong, the translation will also be worse.

Manga profiles

Profiles store manga-specific translation rules. A Frieren profile can preserve names like Frieren, Fern and Himmel. A Witch Hat Atelier profile can preserve terms like Silver Ink, Magic Circles and Brimhats.

Smart Context

Smart Context helps the AI interpret OCR text using manga-specific rules, recurring terminology and style guidance. It is especially useful for fantasy series, power systems, titles, ranks, names and technical worldbuilding.

Batch translation

Batch mode translates all supported images in a folder. Keep file names ordered as 01.jpg, 02.jpg, 03.jpg for clean output.

Troubleshooting

  • If translation is slow, check whether OCR or AI is the bottleneck.
  • If the model does not respond, verify that LM Studio server is running.
  • If fonts look wrong, choose a font that supports the target language.
